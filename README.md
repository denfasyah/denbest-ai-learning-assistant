# AI Learning Assistant (Monorepo Boilerplate)

A context-aware learning assistant to help students understand course materials using Google Gemini.

## Tech Stack
- **Frontend:** React + Vite
- **Backend:** Node.js + Express
- **Database:** MongoDB
- **AI:** Google Gemini API

## Project Structure
- `backend/`: Express API server
- `frontend/`: React single page application
- `docs/`: Documentation and project issues

## Getting Started

### Prerequisites
- Node.js installed
- MongoDB running (local or Atlas)
- Gemini API Key (optional for setup)

### Setup

1. **Clone the repository**
2. **Setup Backend:**
   ```bash
   cd backend
   npm install
   # Configure .env file
   npm run dev
   ```
3. **Setup Frontend:**
   ```bash
   cd frontend
   npm install
   npm run dev
   ```

## Development
- Backend runs on: `http://localhost:5000`
- Frontend runs on: `http://localhost:5173`
